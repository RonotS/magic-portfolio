## Understanding Core Web Vitals

Performance has become a defining factor in modern web development. Google introduced Core Web Vitals as key metrics to measure user experience, focusing on **loading speed, interactivity, and visual stability**.

## Key Metrics

### Largest Contentful Paint (LCP)
LCP measures how quickly the largest visible content (like an image or heading) appears. A good LCP score is under **2.5 seconds**.
